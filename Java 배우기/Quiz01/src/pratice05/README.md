# 문제5

## 문제 설명

n개의 데이터가 height 배열에 주어졌다.  
height 배열의 가각의 원소는 아래 그림과 같이 각 벽에 대한 높이를 의미한다.

이와 같이 높이 값들이 주어졌을 때,  
이 중에서 어떤 두 벽을 고르면 가장 많은 물을 담을 수 있는지를 확인하고  
그떄의 면적을 출력하세요


## 입출력 예시 

| 입력 | 출력 |
| :---: | :---: |
| 1, 8, 6, 2, 5, 4, 8, 3, 7 | 49 |
| 5, 3, 9, 2, 1, 1, 1, 1, 1, 1, 1, 1, 1, 2 | 26 |
