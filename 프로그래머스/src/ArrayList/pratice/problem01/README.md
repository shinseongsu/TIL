# 최댓값 인덱스 구하기

- 주어진 입력중 최대값을 구하고, 최댓값이 이 위치하는 index 값의 목록을 반환하세요.

- 입력:  
[1, 3, 5, 4, 5, 2, 1]

입력된 목록의 최대값은 5입니다.  
5와 동일한 값을 가진 위치는 3번째, 5번째 위치 입니다.  
이 위치에 해당하는 index는 [2,4]입니다. 

출력:  
[2, 4]

---

## 입출력 예 

입력 : [1, 3, 5, 4, 5, 2, 1]  
출력 : [2, 4]

입력 : [3, 6, 10, 1, 7, 2, 4, 6, 10, 9]
출력 : [2, 8]
